# Customer Churn Prediction

Predicting telecom customer churn using Python (Random Forest) and visualizing insights in Power BI. 

## Tech Stack
- Python (Pandas, Scikit-learn, Seaborn)
- Power BI
- Machine Learning & Data Visualization

## Features
- Data cleaning and preprocessing
- Random Forest classifier to predict churn
- Interactive Power BI dashboard with KPI cards, charts, and slicers

## How to Run
1. Open `Python/Customer_Churn_Prediction.ipynb` in Google Colab.
2. Open `PowerBI/Churn_Dashboard.pbix` in Power BI Desktop.
